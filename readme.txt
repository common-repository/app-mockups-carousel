== Description ==
A very simple plugin to display mobile apps portfolio in a Mock-ups  Carousel view.

Plugin create custom post type – App Mock-ups Carousel, add multiple images and settings. Show your work in high resolution, responsive device mock-up using only shortcodes. 

Also work with Gutenberg shortcode block. 

= This plugin contain folowing shortcode: =
<code>[wpos_amc_carousel id="XX"]</code>

= Here is Template code =
<?php echo do_shortcode('[wpos_amc_carousel id="XX"]'); ?> </code>

= Features include =

* Fully responsive
* Touch-enabled Navigation.
* Slider Center Mode
* Fully accessible with arrow key navigation.
* Drag & Drop Order Change
* Slider Autoplay and Speed Interval

= You can use Following parameters with shortcode =

* **ID:** id="1" (Carousel id.this parameter is required.)
* **device:** device="iphpne-white" (ie select device. We have 2 options "iphpne-white" OR "iphpne-black" )

= Important Note =

Image height width should be 273X485 px to work better.

== Installation ==
1. Upload the 'app-mockups-carousel' folder to the '/wp-content/plugins/' directory.
2. Activate the "app-mockups-carousel" list plugin through the 'Plugins' menu in WordPress.
